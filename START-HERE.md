🐧 LINUX TERMINAL PRACTICE KIT
================================
A hands-on practice folder for complete beginners.
Follow along with the weekly LinkedIn & Instagram series.

HOW TO USE THIS KIT
═══════════════════
1. Download and unzip this folder
2. Open your terminal
3. Navigate into the folder:
     cd linux-practice-kit
4. Start with Module 01:
     cd 01-navigation
5. Read the README file:
     cat README.md
6. Follow every exercise in order
7. When done, move to the next module:
     cd ../02-files

FOLDER STRUCTURE
════════════════
linux-practice-kit/

├── 02-files/                 ← touch, cat, cp, mv, rm, nano, head, tail

├── 04-text/                  ← sort, uniq, cut, awk, sed, tee
├── 05-permissions/           ← chmod, chown, sudo, groups
├── 06-archive/               ← tar, zip, unzip
└── 07-pipes-redirect/        ← pipes |, redirect >, >>, &&, keyboard shortcuts

KEYBOARD SHORTCUTS TO LEARN TODAY
═══════════════════════════════════
Ctrl + C   → Cancel any running command
Ctrl + L   → Clear the screen
Ctrl + R   → Search your command history
Tab        → Auto-complete file and folder names
Up Arrow   → Repeat the last command
!!         → Repeat the last command (same as Up Arrow + Enter)
sudo !!    → Repeat last command with admin rights

RULES FOR LEARNING
═══════════════════
1. Type every command manually. Do not copy-paste.
   Your muscle memory learns faster when you type.

2. Make mistakes. They teach you more than success.
   The worst you can do in this practice folder is delete a file.
   If that happens, just re-download the kit.

3. One module per week is the right pace.
   Do not rush. Master one thing, then move to the next.

4. After each exercise, ask: "What if I change this?"
   Experiment. Change the flags. Try different files.
   That is how real learning happens.
