═══════════════════════════════════════════════════════
 KEYBOARD SHORTCUTS — Learn these by heart
═══════════════════════════════════════════════════════

  These are not commands you type — they are key combinations.
  They work in every Linux and Mac terminal.

  ┌─────────────────┬──────────────────────────────────────────┐
  │ Ctrl + C        │ CANCEL the current running command        │
  │ Ctrl + L        │ CLEAR the screen (same as typing: clear)  │
  │ Ctrl + D        │ EXIT the terminal session                  │
  │ Ctrl + R        │ SEARCH your command history interactively  │
  │ Ctrl + Z        │ PAUSE a running process (send to bg)       │
  │ Up Arrow        │ Repeat the LAST command                    │
  │ Down Arrow      │ Go forward in command history              │
  │ Tab             │ AUTO-COMPLETE file/folder names            │
  │ Tab Tab         │ Show ALL possible completions              │
  │ Ctrl + A        │ Jump to START of the line                  │
  │ Ctrl + E        │ Jump to END of the line                    │
  │ Ctrl + W        │ Delete the word BEFORE cursor              │
  │ Ctrl + K        │ Cut everything AFTER cursor                │
  │ Ctrl + U        │ Cut entire line                            │
  │ Ctrl + Y        │ Paste what was cut with Ctrl+K or Ctrl+W  │
  │ !!              │ Repeat the very last command               │
  │ !number         │ Run command #number from history           │
  │ sudo !!         │ Repeat last command WITH sudo              │
  └─────────────────┴──────────────────────────────────────────┘

 EXERCISE 1 — Practice Ctrl+R (history search)
═══════════════════════════════════════════════════════

  Press Ctrl+R
  You will see:   (reverse-i-search)`':

  Start typing a word from a past command.
  It will find the most recent match.
  Press Ctrl+R again to cycle to older matches.
  Press Enter to run the matched command.
  Press Ctrl+C to cancel without running anything.

  Try: Ctrl+R then type "grep"
  Try: Ctrl+R then type "cd"

 EXERCISE 2 — Practice Tab completion
═══════════════════════════════════════════════════════

  Type:   cd ../02
  Then press Tab.
  It will auto-complete to:  cd ../02-files/

  Type:   cat ../02-files/sh
  Press Tab.
  It completes to:  cat ../02-files/shopping-list.txt

  If nothing happens after one Tab, press Tab again.
  It shows ALL possible matches.

 EXERCISE 3 — The pipe  |
═══════════════════════════════════════════════════════

  The pipe takes output from one command and feeds it
  into the next command. Think of it as a connection.

  List files and search the output:
    ls ../02-files/ | grep ".txt"

  Sort the cities and count them:
    cat ../01-navigation/cities.txt | sort | wc -l

  Find top 5 highest scores:
    sort -k2 -n -r ../04-text/scores.txt | head -5

  Long chain — count employees per department:
    cut -d',' -f5 ../04-text/employees.csv | sort | uniq -c | sort -rn

═══════════════════════════════════════════════════════
 EXERCISE 4 — Redirect  >  and  >>
═══════════════════════════════════════════════════════

  Create a new file with output:
    ls ../03-search/reports/ > report-list.txt
    cat report-list.txt

  Overwrite it with different output:
    date > report-list.txt
    cat report-list.txt      (now it only has the date)

  Append instead of overwrite:
    echo "Generated by: $(whoami)" >> report-list.txt
    echo "From folder: $(pwd)" >> report-list.txt
    cat report-list.txt      (now has 3 lines)

  Save grep results to a file:
    grep "ERROR" ../03-search/server.log > errors-only.txt
    cat errors-only.txt

═══════════════════════════════════════════════════════
 EXERCISE 5 — Run multiple commands with  &&
═══════════════════════════════════════════════════════

  && runs the second command ONLY if the first succeeds.

  Create a folder and move into it in one line:
    mkdir test-folder && cd test-folder && pwd

  Create a file and immediately read it:
    echo "Hello World" > hello.txt && cat hello.txt

  Update and upgrade packages (classic combo):
    sudo apt update && sudo apt upgrade

  Cleanup:
    cd .. && rm -r test-folder

═══════════════════════════════════════════════════════
 EXERCISE 6 — Practice !! and history
═══════════════════════════════════════════════════════

  See your command history:
    history

  Run the last command again:
    !!

  Run command number 50 from history:
    !50

  Run last command but with sudo:
    sudo !!

  Search history for a specific word:
    history | grep "tar"
